# Tags

Tags provide a simple way to categorize and filter your traces in Confident's observatory. Unlike metadata, which can contain complex structured data, tags are simple string labels that make it easy to group related traces together, and cannot be applied to spans.

## Add Tags to Traces

Tags are applied at the trace level, making them visible for all spans within that trace.

```python
from deepeval.tracing import observe, update_current_trace

@observe(type="agent")
def agent(query: str, user_id: str):
    tags = []

    # Dynamically add tags
    if "premium" in user_id:
        tags.append("premium-user")
    else:
        tags.append("starter-user")

    update_current_trace(tags=tags)
    return "Agent Response"

agent("Test Tags", "premium-123")
```
