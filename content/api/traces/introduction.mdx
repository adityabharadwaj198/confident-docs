# Evals API for Tracing

You can ingest and query traces in your projects using the `/v1/traces` endpoint, as well as run evals on all of them at realtime. This is especially useful for those not using Python or Type/Javascript, but want to levareage Confident AI on evals on all your traces.

## Tracing on the UI

You can view all traces associated with the project your API key has been logged to by logging into your Confidnet AI account and going to the project anytime.
