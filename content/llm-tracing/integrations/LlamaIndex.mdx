# LlamaIndex

[LlamaIndex](https://www.llamaindex.ai/) is a framework that makes it easy to build production agents that can find information, synthesize insights, generate reports, and take actions over the most complex enterprise data.

## Quickstart

Confident AI provides a `LlamaIndexEventHandler` that can be used to trace LlamaIndex **events.**

Install the following packages:

```bash
pip install -U deepeval llama-index
```

Login with your API key and configure DeepEval as an event handler for LlamaIndex:

```python filename="main.py" showLineNumbers {6, 9}
import llama_index.core.instrumentation as instrument
from deepeval.integrations import LLamaIndexEventHandler
import deepeval

# Login
deepeval.login_with_confident_api_key("<your-confident-api-key>")

# Let DeepEval collect traces
instrument.get_dispatcher().add_event_handler(LLamaIndexEventHandler())
```

Now whenever you use LlamaIndex, DeepEval will collect LlamaIndex event as traces and publish them to Confident AI.

<Callout type="info">
[Event](https://docs.llamaindex.ai/en/stable/module_guides/observability/instrumentation/) in LlamaIndex represents a single moment in time that a certain occurrence took place within the execution of the applicationâ€™s code.
</Callout>

## Example Agent

The example agent will be able to perform calculations using the `multiply` tool.

```python showLineNumbers filename="main.py" {12, 13, 22}
import os
import asyncio
import time
from llama_index.core.agent.workflow import FunctionAgent
from llama_index.llms.openai import OpenAI
import llama_index.core.instrumentation as instrument

from deepeval.integrations import LLamaIndexEventHandler
import deepeval

# Don't forget to setup tracing
deepeval.login_with_confident_api_key("<your-confident-api-key>")
instrument.get_dispatcher().add_event_handler(LLamaIndexEventHandler())


os.environ["OPENAI_API_KEY"] = "<your-openai-api-key>"

def multiply(a: float, b: float) -> float:
    """Useful for multiplying two numbers."""
    return a * b

agent = FunctionAgent(
    tools=[multiply],
    llm=OpenAI(model="gpt-4o-mini"),
    system_prompt="You are a helpful assistant that can perform calculations.",
)
```

Finally, try calling your agent to see your traces on Confident AI:

```python filename="main.py" {4}
from deepeval.tracing import observe
...

@observe()
async def main():   
    response = await agent.run(
        "What's 7 * 8?"
    )
    print(response)

if __name__ == "__main__":
    asyncio.run(main())
    time.sleep(12)
```

```bash
python main.py
```

You can directly view the traces in the **Observatory** by clicking on the link in the output show in the console:

<Callout>
As the agent is running, it will export **events** to the `dispatcher`. The `dispatcher` will send the events to the `LlamaIndexEventHandler` which will convert the events to the appropiate **spans**, create a **trace** and send it to the [Observatory](https://documentation.confident-ai.com/llm-tracing/introduction).
</Callout>
